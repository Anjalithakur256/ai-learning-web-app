# AI Learning Guide Web App

A static web app that provides AI learning roadmaps, topics, quizzes, and user flows.

## Pages
- Home: index.html
- Login: login.html
- Dashboard: dashboard.html
- Profile: profile.html
- Topic: topic.html
- Quiz: quiz.html
- About: about.html
- Privacy: privacy.html

## Local preview
You can serve the site with any static file server. For example:

python -m http.server 8000

Then open http://localhost:8000 in your browser.

## Project structure
- assets/: CSS and JavaScript
- data/: JSON content for roadmaps, topics, and quizzes
- firebase.json, firestore.rules: Firebase configuration files
