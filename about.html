📚 About AI Learning Guide

AI Learning Guide is a structured and interactive web platform designed to help learners master Artificial Intelligence step-by-step. The platform provides curated roadmaps, interactive quizzes, real-time progress tracking, an AI-powered chat assistant, and a Socratic tutor that guides students through problem-solving rather than just giving answers.

AI topics are organized into progressive tracks covering AI Basics, Machine Learning, Deep Learning, NLP, and Generative AI — each structured to move from foundational understanding to real-world implementation.

The platform uses Firebase for authentication and real-time database, OpenRouter as the AI gateway routing to GPT (openai/gpt-oss-20b) for intelligent responses and text-embedding-3-small for RAG-powered context retrieval. It is deployed on Vercel with serverless API proxies.

⚡ Core Features

🗺️ Structured AI Roadmaps

Follow progressive learning paths from AI fundamentals to advanced generative AI concepts.

❓ Interactive Quizzes

Concept-based quizzes with instant feedback, score tracking, and performance history.

📊 Real-Time Progress Tracking

Monitor quiz scores, topic completion, and class-wide performance via live dashboards powered by Chart.js.

🤖 AI Chat Assistant

Powered by GPT (openai/gpt-oss-20b) via OpenRouter with RAG retrieval for context-aware, accurate answers.

🧠 Socratic Tutor

A guided tutor that asks leading questions to help students reason through AI concepts independently.

📷 OCR Homework Helper

Upload a photo of a handwritten or printed problem — the OCR engine extracts the text and sends it through the AI pipeline.

📚 RAG Pipeline

Retrieval-Augmented Generation using text-embedding-3-small embeddings to ground AI responses in course material.

🎓 Teacher / Admin Panel

Real-time student roster, score analytics, and class performance charts exclusively for faculty accounts.

🎯 Vision

To make AI education structured, practical, and accessible for every student who wants to build real-world AI skills — guided by intelligent tutoring and honest progress data.
